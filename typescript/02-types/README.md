# Typescript 
## 02-types 
**Types**
- number for values: 1, -10, 5.3
  - all numbers; no diffrence between integers or floats
- string for values: 'hi' "HI" \`hi\`
  - all text values
- boolean for values: true, false
  - just these two
- object for javascript objects: {age: 30}
  - any javascript object
